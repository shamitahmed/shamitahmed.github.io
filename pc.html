🐥Compadre - Agentic Companion

    Compadre is a real-time, multimodal AI system that generates a fully interactive, reactive stream audience for single-player games. Leveraging vision models to analyze live gameplay and speech-to-text to process player audio, a custom Python backend orchestrates a diverse cast of LLM-driven personas. These agents dynamically respond to the game state with contextual chat, TTS audio, and simulated donations, all rendered through a low-latency WebSocket overlay built in Electron.



    Genre: AI Companionship
    Role: Design, Programming

    Supercell X Florent Agentic Frontier Hackathon 2026
