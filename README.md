# Shamit Ahmed — Personal Portfolio

🌐 **Live site:** [shamitahmed.github.io](https://shamitahmed.github.io/)

Personal portfolio website for **Shamit Ahmed**, a multidisciplinary programmer with 4 years of game industry experience from Bangladesh, currently based in Finland.

---

## About

This is a static HTML/CSS/JS website hosted on GitHub Pages. It showcases Shamit's work across game development, XR (VR/AR), and research.

## Sections

- **About** — Background and bio
- **Projects** — Featured games and interactive experiences
- **Publications** — Research papers and academic work
- **Experience** — Education and career history

## Tech Stack

- HTML, CSS, JavaScript
- [Bootstrap](https://getbootstrap.com/)
- [AOS](https://michalsnik.github.io/aos/) (Animate on Scroll)
- [Google Model Viewer](https://modelviewer.dev/) (3D model rendering)
- Hosted via **GitHub Pages**
